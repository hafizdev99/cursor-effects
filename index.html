KURULUM INSTALLATION

Nasıl Yüklenir? How to Install?

Chrome Web Store'a gerek yok. 5 adımda kurulum tamamlanır. No Chrome Web Store needed. Installation completes in 5 steps.

01
ZIP dosyasını indir Download the ZIP file
Yukarıdaki "İndir" butonuna tıklayın. ZIP dosyası otomatik olarak bilgisayarınıza inecek. Click the "Download" button above. The ZIP file will automatically save to your computer.
02
ZIP'i çıkart Extract the ZIP
İndirilen ZIP dosyasına sağ tıklayıp "Tümünü çıkart" seçeneğini kullanın. Klasörü hatırlayacağınız bir yere çıkartın. Right-click the downloaded ZIP and select "Extract All". Extract the folder to a location you'll remember.
03
Chrome Extension sayfasını aç Open Chrome Extensions page
Chrome adres çubuğuna chrome://extensions yazıp Enter'a basın. Type chrome://extensions in your Chrome address bar and press Enter.
04
Geliştirici modunu aç Enable Developer Mode
Sayfanın sağ üst köşesindeki Geliştirici modu toggle'ını açık konuma getirin. Toggle the Developer mode switch in the top-right corner of the page to the ON position.
05
Klasörü yükle ve kullanmaya başla Load the folder and start using
Paketlenmemiş öğe yükle butonuna tıklayın ve çıkarttığınız klasörü seçin. Araç çubuğundaki Cursor Effects ikonuna tıklayarak efektinizi seçin! 🎉 Click Load unpacked and select the extracted folder. Click the Cursor Effects icon in your toolbar and choose your effect! 🎉
